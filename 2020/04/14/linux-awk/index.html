1. 用法：

awk [选项参数] ‘script’ var=value file(s)

2. 内建变量

变量 描述
$0 完整的输入记录
NF 一条记录的字段的数目
NR 已经读出的记录数，就是行号，从1开始
FS 字段分隔符(默认是任何空格)

3. 正则

字符串匹配 输出第二列包含 “th”，并打印第二列与第四列

  • awk ‘$2 ~ /th/ {print $2,$4}’ log.txt

  • ~ 表示模式开始。// 中是模式。

输出包含”re” 的行

  • awk ‘/re/ ‘ log.txt

模式取反

  • awk ‘$2 !~ /th/ {print $2,$4}’ log.txt

  • awk ‘!/th/ {print $2,$4}’ log.txt

4. awk脚本

关于awk脚本，我们需要注意两个关键词BEGIN和END。

  • BEGIN{ 这里面放的是执行前的语句 }
  • END {这里面放的是处理完所有的行后要执行的语句 }
  • {这里面放的是处理每一行时要执行的语句}

5. 示例

每行按空格或TAB分割，输出文本中的1、4项

  • awk ‘{print $1,$4}’ log.txt

  • awk ‘{printf “%-8s %-10s\n”,$1,$4}’ log.txt # 格式化输出

awk -F #-F相当于内置变量FS, 指定分割字符

  • awk -F, ‘{print $1,$2}’ log.txt

运算符

  • awk ‘$1==2 {print $1,$3}’ log.txt

内建变量

  • awk ‘BEGIN{printf “%4s %4s %4s %4s %4s %4s %4s %4s %4s\n”,”FILENAME”,”ARGC”,”FNR”,”FS”,”NF”,”NR”,”OFS”,”ORS”,”RS”;printf “———————————————\n”} {printf “%4s %4s %4s %4s %4s %4s %4s %4s %4s\n”,FILENAME,ARGC,FNR,FS,NF,NR,OFS,ORS,RS}’ log.txt

  • 最大值

  • cat data|awk ‘BEGIN {max = 0} {if ($1>max) max=$1 fi} END {print “Max=”, max}’

  • 最小值

  • min的初始值设置一个超大数即可

  • 平均

  • 求平均cat data|awk ‘{sum+=$1} END {print “Average = “, sum/NR}’

  • 求和

  • 求和cat data|awk ‘{sum+=$1} END {print “Sum = “, sum}’

6. 原理

  • 1、通过关键字 BEGIN 执行 BEGIN 块的内容，即 BEGIN 后花括号 {} 的内容，只执行一次。
  • 2、完成 BEGIN 块的执行，开始执行body块。
  • 3、读入有 \n 换行符分割的记录。
  • 4、将记录按指定的域分隔符划分域，填充域，$0 则表示所有域(即一行内容)，$1 表示第一个域，$n 表示第 n 个域。
  • 5、依次执行各 BODY 块，pattern 部分匹配该行内容成功后，才会执行 awk-commands 的内容。
  • 6、循环读取并执行各行直到文件结束，完成body块执行。
  • 7、开始 END 块执行，END 块可以输出最终结果。
